# themectl Quickstart

## Install

```bash
./install.sh
```

## Validate Installation

```bash
./themectl validate
```

## Apply a Built-in Theme

```bash
./themectl theme apply mocha
```

## Generate From Image

```bash
./themectl generate --image ~/Pictures/Wallpapers/example.jpg --apply
```
